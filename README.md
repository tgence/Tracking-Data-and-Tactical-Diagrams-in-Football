# Tracking Data & Tactical Diagrams in Football

An interactive tool for exploring football tracking data, visualising tactical movements, drawing annotations, and simulating hypothetical scenarios.

## Features

- Load full matches (tracking + events)
- Interactive pitch with player icons, orientations, offside line and speed vectors
- Timeline navigation with event markers (goals, cards, substitutionsâ€¦)
- Drawing tools: passes, runs, dribbles, zones
- Simulation mode: create what-if tactical scenarios
- Auto camera modes (ball tracking, set-piece presets)
- Two visual themes (Classic / Black & White)
